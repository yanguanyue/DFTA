# HAM10000 Segmentation Training and Mixed Data Experiments

This directory contains:
- **UNet** and **SegFormer** training code for real and mixed datasets
- Mixed dataset construction scripts (see `metirc/classifier/mix_synthetic.py`)
- Evaluation metrics: mDice and mIoU

## Project Structure

```
metirc/segmentation/
├── train_segmentation.py         # Main training script
├── run_experiments.py            # Batch experiment runner
├── run_4sets_512.py              # Train 4 sets with 512x512 resolution
├── mix_synthetic_seg.py          # Mixed dataset construction
├── dataset.py                    # Dataset classes
├── models.py                     # Model definitions
├── metrics.py                    # Evaluation metrics
├── summarize_segmentation.py     # Results summarization
├── run_generated_models.py       # Train on generated models
├── requirements.txt              # Dependencies
└── README.md                     # This file
```

## Dataset Structure Conventions

### Real Dataset (HAM10000 Original Structure)
```
/data/HAM10000/input/
  train/
    HAM10000_img_class/<class>/*.jpg
    HAM10000_seg_class/<class>/*_segmentation.png
  val/
    HAM10000_img_class/<class>/*.jpg
    HAM10000_seg_class/<class>/*_segmentation.png
```

### Mixed Dataset (Generated by This Script)
```
<output_root>/
  train/
    images/<class>/*.png
    masks/<class>/*.png
  val/
    images/<class>/*.png
    masks/<class>/*.png
```

## 1) Build Mixed Dataset

Example (using Siamese generated data):
```bash
python metirc/segmentation/mix_synthetic_seg.py \
  --real-root /root/autodl-tmp/data/HAM10000/input \
  --synthetic-root /root/autodl-tmp/data/output/Siamese \
  --output-root /root/autodl-tmp/data/mixed_segmentation/Siamese \
  --minority-cap 1500 \
  --majority-cap 500
```

## 2) Training and Evaluation

### Train on Real Dataset
```bash
python metirc/segmentation/train_segmentation.py \
  --dataset-root /root/autodl-tmp/data/HAM10000/input \
  --data-layout ham10000 \
  --model unet \
  --epochs 50 \
  --batch-size 8 \
  --image-size 256
```

### Train on Mixed Dataset
```bash
python metirc/segmentation/train_segmentation.py \
  --dataset-root /root/autodl-tmp/data/mixed_segmentation/Siamese \
  --data-layout mixed \
  --model segformer \
  --epochs 50 \
  --batch-size 8 \
  --image-size 256
```

Training logs output mDice and mIoU for each epoch, and save the best model with `metrics.json` in the `--save-dir` directory.

## 3) Experiment Summary (Optional)

You can use `run_experiments.py` to batch run experiments on real and mixed datasets and automatically summarize metrics:
```bash
python metirc/segmentation/run_experiments.py \
  --real-root /root/autodl-tmp/data/HAM10000/input \
  --mixed-root /root/autodl-tmp/data/mixed_segmentation/Siamese \
  --output-csv /root/autodl-tmp/output/segmentation/summary/metrics_summary.csv
```

## 4) Train 4 Sets (512×512)

This script trains 4 groups of **Real/Siamese/Siamese-update/flow** sequentially and outputs a summary JSON. If the corresponding `*_metrics.json` already exists, that task will be skipped.
Training processes are launched in the background, with logs written to each `save_dir/log.txt`.
```bash
python metirc/segmentation/run_4sets_512.py \
  --real-root /root/autodl-tmp/data/HAM10000/input \
  --mixed-root /root/autodl-tmp/data/mixed_segmentation \
  --output-json /root/autodl-tmp/output/segmentation/outputs/metrics_4sets_512.json \
  --max-steps 15000 \
  --val-interval 3000 \
  --batch-size 4 \
  --image-size 512
```

## Supported Models

- **UNet**: Standard U-Net architecture for semantic segmentation
- **SegFormer**: Transformer-based segmentation model

## Evaluation Metrics

- **mDice**: Mean Dice coefficient
- **mIoU**: Mean Intersection over Union
